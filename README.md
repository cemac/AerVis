## Run Example

#### Python library
```
from aervis import *

dataclass = AerData('bk417a')
# This creates a netCDF using the UKCA .pp files. If this already exists, we load the created netCDF instead 

```
#### CommandLine
``` ./aervis --name='bk417a' --loc='./' --stash='STASHMASTER|file2|file3 ```
